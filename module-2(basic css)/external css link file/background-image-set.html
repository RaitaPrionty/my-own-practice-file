COMPUTER

The computer is one of the greatest inventions of modern science. It was invented by Charles Babbage. It is an electronic machine. It can store a large amount of data. The computer has made a revolution in all fields of human activities. It is widely used in schools, colleges, banks, hospitals, railways, airlines, hotels and so on. By using a computer, even the biggest work can be done in less time. Computer helps the students to learn new things. Scientists and scholars can make use of the computer in their studies and research work. While it has numerous benefits, it can also create health problems if we get addicted to it. So, we should use it wisely
